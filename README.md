# Invoice App

A clean and responsive invoice management app built with React and Tailwind CSS.

Users can create, edit, delete, and filter invoices, with automatic calculations and persistent storage.

---

## ✨ Features

- Create, edit, and delete invoices
- Filter invoices by status (Draft, Pending, Paid)
- Dynamic item list with auto total calculation
- Payment terms with automatic due date calculation
- Form validation with clear error states
- Dark mode toggle
- Responsive design (mobile → desktop)
- LocalStorage persistence (data is saved in browser)

---

## 🧱 Built With

- React
- React Router
- Tailwind CSS
